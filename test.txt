git 簡単な使い方
1. git init //リポジトリの作成（最初の一回だけ！）
2. git add . //で保存したいファイルのステージング
3. git commit -m "何をしたのかメッセージ" //（実際に保存）
4. 確認したい時は、git log

で、2と3を基本的には繰り返していく。


ブランチ操作について
1. git branch ブランチ名 (ブランチをきる)
2. git checkout 移動したい先のブランチ名（ブランチ間の移動）
3. 移動した先で、コードの編集や作業をする。
4. 編集したファイルのステージング(git add .)
5. コミット(git commit -m "何をしたのかメッセージ")
6. git checkout 移動したい先のブランチ名(masterとか)
7. git merge 編集を終えて、取り込みたいブランチ名
8. git branch -d 削除したいブランチ名（ブランチの削除）
